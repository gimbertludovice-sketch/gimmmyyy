## 🎨 Project Overview
This repository contains a fully responsive, modern web portfolio featuring a dark/light cyber-navy aesthetic. It is structured cleanly using single-file component architecture for ease of deployment.

### Key Features
- **Responsive Navigation:** Interactive mobile menu with smooth transitions.
- **Dynamic UI Windows:** Modal popups for detailed project views and credentials.
- **Scroll Reveals:** Elements smoothly animate into view using `IntersectionObserver`.
- **Active Navigation Tracking:** Automatically highlights the current section as you scroll.
- **Interactive Tech Stack:** Hover animations displaying language proficiencies.

## 🛠️ Built With
- **HTML5** - Semantic structure
- **CSS3** - Custom properties (CSS variables), Flexbox, Grid, and smooth keyframe animations
- **JavaScript (ES6+)** - Vanilla DOM manipulation and UI logic
- **Google Fonts** - Space Grotesk & JetBrains Mono for a sleek, developer-themed typography
